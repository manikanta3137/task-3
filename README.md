# Exploratory Data Analysis (EDA) on Student Placement & Career Success Dataset

## 📊 Project Overview

This project performs Exploratory Data Analysis (EDA) on a Student Placement & Career Success Dataset to uncover patterns, trends, and factors that influence student placement outcomes.

EDA helps transform raw data into meaningful insights using statistical summaries, visualizations, and correlation analysis. The findings can help students, educators, and placement coordinators understand the key drivers of career success.

---

## 🎯 Objectives

* Explore and understand the dataset structure.
* Perform data cleaning and preprocessing.
* Generate statistical summaries.
* Analyze feature distributions.
* Identify correlations among variables.
* Discover factors affecting placement outcomes.
* Present findings through visualizations and insights.

---

## 📂 Dataset Information

**Dataset:** Student Placement & Career Success Dataset

### Dataset Characteristics

* Total Records: 25,000
* Total Features: 44
* Data Type: Mixed (Numerical and Categorical)

### Key Attributes

* CGPA
* Aptitude Score
* Communication Skills
* Resume Score
* Projects Completed
* Internships Completed
* Certifications
* Technical Skills
* Attendance Percentage
* Placement Outcome
